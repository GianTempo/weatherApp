How does it work?

This app gets the weather data from the OpenWeatherMap API. It also gets your location trough the HTML'S geolocation API (visit W3's Geolocation API page for more info). This is the reason why the app asks you to allow the access to your location.

You can also check for the weather on an especific city by writing it's name in the "City Name" input. The "Country Code" input is optional, refer to ISO 3166 for the country codes.

You can also select the degree unit you want to use between three options: Celsius (default), Farenheit or Kelvin.
